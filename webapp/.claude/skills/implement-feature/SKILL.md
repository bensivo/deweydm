Implement the feature given in the feature-plan md document. 

For styling, read the relevant style guides in `docs/style-guides`, so you're in compliance with the standard styles. If the existing codebase doesn't match, use the style-guide anyways. 

DO NOT try to run any terminal commands to build code or run tests, just do the implementation and tell the user to build the app after.
